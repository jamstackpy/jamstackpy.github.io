Introduction

Jamstackpy allows you to generate static websites using all the power of Flask, so you can modify the files as much as you like to achieve different results and create fantastic sites.

Install Jamstackpy

Templates

Jamstackpy has default templates such as Massively and Phantom, both courtesy of HTML5UP, however as well as being able to customise these, you can create your own templates, and even include them in the templates supported by Jamstackpy.

To find out how to contribute a template please see this article from the official wiki.
